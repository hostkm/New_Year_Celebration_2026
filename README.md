## ğŸ§  About the Project

**New_Year_Celebration_2026** is a **pure Python terminal application** that simulates a New Year countdown and celebration:

- Displays a **big ASCII year `2025 â†’ 2026`**
- Shows a **live simulated clock** from `23:57:00` to `00:03:00`
- Animates the **digit transition**:
  - `5` moves away
  - `6` smoothly appears
- Triggers celebration **exactly at midnight**
- Displays **festive messages, emojis, stars, and confetti**
